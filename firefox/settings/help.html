Extension Help & Documentation

How to Use the Extension

  • Click Edit to customize links on your new tab page.
  • Edit link name, URL, and icon by clicking on any link card.
  • Your changes are automatically saved in the browser.

Troubleshooting

If something doesn’t work, try reloading the extension from about:addons and clicking “Reload”.

How to Set This Extension as Firefox Homepage

Firefox does not allow extensions to directly modify your homepage settings. But you can set the extension as your homepage manually by following these steps:

  1. Install or load the extension using about:debugging or via the Add-ons menu.
  2. Open a new tab — your custom extension should appear.
  3. Right-click anywhere on the extension and choose “Copy Page URL”, or copy it from the address bar.
  4. The URL will look like this: moz-extension://abc123-4567-defg/main.html
  5. Go to Firefox Settings > Home.
  6. Under “Homepage and new windows”, select Custom URLs.
  7. Paste the copied URL into the box.

Done! Now every time you open Firefox, it will show your custom extension as the homepage.
